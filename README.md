# 📊 Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes sales data using Microsoft Excel and creates an interactive dashboard using pivot tables, charts, and slicers.

## 🛠 Tools Used
- Microsoft Excel

## 📊 Analysis Performed
- Sales by Region
- Sales by Category

## 📈 Key Insights
- West region has the highest sales
- East region is the second highest contributor
- South region has the lowest sales
- Technology category performs best

## 🎯 Outcome
Created an interactive dashboard to help understand business performance and support decision-making.
